Initial Release

A lightweight, bloat-free background clipping tool for Windows.

Designed for minimal performance impact, the application runs as a single executable and continuously buffers gameplay in the background. Simply select the monitor(s) you want to capture, start the service, and press Ctrl+Shift+C at any time to save the last 30 seconds of gameplay.

Features

- Lightweight and easy to use
- Single executable — no complicated setup
- Multi-monitor support
- Global hotkey clipping (Ctrl+Shift+C)
- Continuous background replay buffer
- Clips saved directly to your Videos folder
- Built for gamers who want quick highlights without heavyweight recording software

This is the first public release. Feedback, bug reports, and feature suggestions are welcome.
